# User Manual - Smart Campus

## 1. Introduction
Welcome to the Smart Campus platform. This system integrates academic management, attendance, cafeteria services, and campus events into one digital solution.

## 2. Roles & Login
- **Students:** Login with Student Email (e.g., `student@campus.edu`).
- **Faculty:** Login with Faculty Email.
- **Admin:** Login with Admin Email for full access.
- **Default Password:** `Password1` (for demo accounts).

## 3. Student Guide
- **Attendance:**
    - View QR Code in Mobile/Dashboard. Scanners will scan this code.
    - Check Attendance percentage in "Academics" > "Attendance".
- **Cafeteria:**
    - Top-up Wallet in "Wallet" page.
    - Reserve meals in "Cafeteria" page.
    - Show Reservation QR at the counter.
- **Events:**
    - Browse upcoming events and click "Register".

## 4. Faculty Guide
- **Manage Courses:** View enrolled students and sections.
- **Attendance:**
    - Start a new Session -> Generates a Session Code/QR.
    - Monitor real-time check-ins.
    - Manually correct attendance records.

## 5. Admin Guide (New in Part 4)
- **Dashboard:**
    - View live stats: Active users, Attendance rates, Meal usage.
- **Analytics:**
    - **Academic:** GPU distribution, At-risk students.
    - **Meal:** Daily cafeteria revenue and reservation counts.
    - **IoT:** Monitor physical campus sensors (Temp, Occupancy).
- **Notifications:**
    - Send broadcast announcements.
    - Configure system-wide alerts.

## 6. Notification Settings
Navigate to "Settings" > "Notifications" to toggle Email/SMS/Push preferences for different categories (Academic, Social, System).

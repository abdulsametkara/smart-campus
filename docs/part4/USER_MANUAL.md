# Smart Campus - User Manual (Complete)

## Introduction

Welcome to Smart Campus - your all-in-one digital university platform. This guide covers all features for Students, Faculty, and Administrators.

---

## 1. Getting Started

### 1.1 Registration

1. Navigate to the Registration page
2. Fill in your details:
   - Full Name
   - Email (use your university email)
   - Password (min 8 chars, 1 uppercase, 1 number)
   - Role: Student or Faculty
3. Click "Register"
4. Check your email for verification link
5. Click the link to activate your account

### 1.2 Login

1. Enter your email and password
2. If 2FA is enabled, enter the 6-digit code from your authenticator app
3. Click "Login"

### 1.3 Forgot Password

1. Click "Forgot Password" on login page
2. Enter your email
3. Check inbox for reset link
4. Create a new password

## 2. Student Guide

### 2.1 Dashboard

Your personalized dashboard shows:
- Today's schedule
- Upcoming deadlines
- Attendance summary
- Wallet balance
- Notifications

### 2.2 Course Enrollment

1. Go to **Academic → Course Catalog**
2. Browse available courses
3. Click on a course to view sections
4. Select a section and click "Enroll"
5. System checks:
   - Prerequisites completed
   - No schedule conflicts
   - Available capacity
6. If advisor approval required, wait for confirmation

### 2.3 Viewing Grades

1. Navigate to **Academic → My Grades**
2. View grades by course
3. Check GPA calculation
4. Download transcript (PDF) from the Transcript page

### 2.4 Attendance

**Checking In:**
1. Go to **Attendance → Check In**
2. If QR session active: Scan the QR code displayed by instructor
3. If GPS session: Your location is verified automatically
4. Confirmation appears on success

**Viewing Attendance:**
1. Go to **Attendance → My Attendance**
2. See attendance percentage per course
3. ⚠️ Warning appears if below 70%

**Submitting Excuses:**
1. Go to **Attendance → Submit Excuse**
2. Select the missed session
3. Write reason and upload document (if any)
4. Submit for instructor review

### 2.5 Cafeteria & Meals

**Adding Money to Wallet:**
1. Go to **Wallet**
2. Click "Add Balance"
3. Enter amount
4. Complete payment

**Reserving Meals:**
1. Go to **Cafeteria → Menu**
2. Browse weekly menu
3. Select a meal and click "Reserve"
4. Balance is deducted automatically
5. QR code appears in "My Reservations"

**Using Reservation:**
1. At cafeteria, show your QR code
2. Staff scans it
3. Meal is served!

### 2.6 Events

1. Go to **Events**
2. Browse upcoming events
3. Click an event for details
4. Click "Register"
5. For paid events, balance is deducted
6. QR code appears for check-in

### 2.7 Notifications

- Bell icon in navbar shows unread count
- Click to see recent notifications
- Go to **Settings → Notifications** to configure preferences

## 3. Faculty Guide

### 3.1 Managing Courses

1. Go to **Academic → My Sections**
2. View sections you're teaching
3. See enrolled students
4. Access gradebook

### 3.2 Taking Attendance

**Starting a Session:**
1. Go to **Attendance → Start Session**
2. Select your section
3. Choose method:
   - **QR Code**: Students scan displayed code
   - **GPS**: Students must be within radius
4. Set duration (e.g., 15 minutes)
5. Click "Start"
6. Monitor real-time check-ins

**Manual Correction:**
1. Go to **Attendance → Session History**
2. Select a past session
3. Click on a student's record
4. Change status (Present/Absent/Late/Excused)
5. Save changes

### 3.3 Grading

**Creating Exams:**
1. Go to **Grades → Exams**
2. Click "Create Exam"
3. Fill in: Name, Type (Midterm/Final/Quiz), Weight, Date
4. Save

**Entering Grades:**
1. Go to **Grades → Enter Grades**
2. Select exam
3. Enter scores for each student
4. Click "Submit Grades"
5. Students are notified automatically

**Viewing Reports:**
1. Go to **Grades → Reports**
2. See grade distribution
3. Export to Excel

### 3.4 Reviewing Excuses

1. Go to **Attendance → Excuse Requests**
2. View pending requests
3. Review attached documents
4. Approve or Reject with comment

### 3.5 Advising (if applicable)

1. Go to **Academic → My Advisees**
2. View assigned students
3. Approve/Reject enrollment requests
4. Monitor academic progress

## 4. Admin Guide

### 4.1 Dashboard

The Admin Dashboard provides:
- Total users, courses, enrollments
- Today's active users
- Attendance rate
- Meal reservations
- System health

### 4.2 User Management

1. Go to **Admin → Users**
2. Search and filter users
3. Click a user to:
   - Edit profile
   - Change role
   - Reset password
   - Deactivate account

### 4.3 Course Management

1. Go to **Admin → Courses**
2. Create new courses with prerequisites
3. Create sections and assign instructors
4. Set capacities

### 4.4 Analytics

**Academic Performance:**
- GPA distribution by department
- At-risk students (GPA < 2.0)
- Pass/Fail rates

**Attendance:**
- Overall attendance rates
- Courses with low attendance
- Students with critical absences

**Cafeteria:**
- Daily meal counts
- Revenue reports
- Peak hours

**Events:**
- Popular events
- Registration vs Check-in rates

**Exporting:**
1. Navigate to any Analytics page
2. Click "Export" button
3. Choose format: Excel, PDF, or CSV

### 4.5 Announcements

1. Go to **Admin → Announcements**
2. Click "Create Announcement"
3. Write title and content
4. Select audience (All/Students/Faculty)
5. Publish

### 4.6 Notifications

**Broadcast:**
1. Go to **Admin → Notifications**
2. Click "Send Notification"
3. Write message
4. Select recipients (All or specific users)
5. Send

### 4.7 IoT Monitoring (Bonus)

1. Go to **Admin → IoT Dashboard**
2. View live sensor readings
3. Check temperature, occupancy, energy usage
4. Historical charts available

### 4.8 System Settings

1. Go to **Admin → Settings**
2. Configure:
   - Enrollment periods
   - Attendance thresholds
   - Email templates

## 5. Settings & Security

### 5.1 Enable 2FA

1. Go to **Profile → Security**
2. Click "Enable 2FA"
3. Scan QR code with Google Authenticator
4. Enter verification code
5. Save backup codes

### 5.2 Change Password

1. Go to **Profile → Security**
2. Enter current password
3. Enter new password (twice)
4. Save

### 5.3 Notification Preferences

1. Go to **Settings → Notifications**
2. Toggle Email/Push/SMS for each category
3. Save preferences
